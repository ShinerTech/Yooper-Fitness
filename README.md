# Yooper Fitness Website

This project contains the static HTML source files for the Yooper Fitness website. It has been configured with relative paths so that it can be hosted on GitHub Pages or run locally without special server configurations.

## Running Locally

To run the site locally, navigate to this folder in your terminal and start a basic HTTP server:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000/index.htm` in your browser.

*(Note: If you run your server from a different directory, navigate to the `Yooper Fitness` folder path in your browser).*
